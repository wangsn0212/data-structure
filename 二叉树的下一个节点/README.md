## 任务 ##
给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。

## 关键思路 ##

题目给出了pNode.next指向给定节点的父节点，这样while循环遍历到pNode.next == null，此时的pNode指向的就是根节点。所以本题只需要给出当前节点即可，根节点可以使用next求出来。


          a
       b      c
     d   e   f  g
        h i  
    中序遍历：[d,b,h,e,i,a,f,c,g]
    1. 如果一个节点有右子树，那么他的下一个节点就是他的右子树中的最左节点，
    2.1 如果一个节点是他父节点的 左子节点，那么他的下一个节点就是他的父节点。  
    2.2 如果一个节点没有右子树，且还会父节点的右子节点，则沿着父节点的指针一直向上遍历，知道找到一个是它父节点的左子节点的节点，若存在，该节点 的父节点则为下一点。  
    其中2.1和2.2和合并为查找父节点
